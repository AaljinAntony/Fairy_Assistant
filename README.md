## Overview
Fairy Assistant is a private, local-first AI companion designed to bridge your desktop and mobile environments. It leverages local LLMs and cross-platform automation to act as a unified digital assistant.

## Key Features
-   **Local Intelligence**: Powered by **Ollama** (Llama 3, etc.) for privacy-preserving inference.
-   **Voice Interface**: Real-time Speech-to-Text using **OpenAI Whisper**.
-   **Long-term Memory**: RAG (Retrieval-Augmented Generation) pipeline using **ChromaDB** with recursive chunking for context retention.
-   **Cross-Platform control**:
    -   **Linux Desktop**: App management, system control, and keyboard automation.
    -   **Android Companion**: Mobile app (Flutter) for remote voice commands and phone automation (Calls, SMS, WhatsApp).

## Tech Stack
-   **Backend**: Python (Flask, SocketIO)
-   **Mobile**: Flutter (Dart)
-   **AI/ML**: LangChain, SentenceTransformers, ChromaDB
-   **Automation**: PyAutoGUI, custom Android intents
